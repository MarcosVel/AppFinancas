# App Finanças

• Aplicativo de controle de finanças.

## Rodando localmente

Instale as dependências

```bash
  yarn
```

Abra o emulador e rode o projeto

```bash
  npx expo start
```


## Stack utilizada

• Expo

• Styled Components

• Firebase

• Date-fns

• React Native Picker

• Async Storage

• Drawer Navigation

• Stack Navigation
